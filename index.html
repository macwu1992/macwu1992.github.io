算法导论 chp2（排序算法）

2016-12-25
算法算法入门
算法, 读书笔记

2.1 插入排序

在书中的插入排序描述如下：
输入：n个数（a1，a2，a3，…，an）
输出：输入序列的一个排列（即重新排序）（a’1，a’2，a’3，…，a’n）

##################################################
pseudo-code

##################################################
INSERTION-SORT
for j->2 to length(A)
do key <- A[j]
//Insert A[j] into the sorted sequence A[1,…,j-1]
i <- j-1
while i>0 and A[i]>key
do A[i+1] <- A[i]
i <- i-1
A[i+1] <- key

##################################################
java-code

##################################################
package com.macwu.chp1;

/**

  • Created by Tong on 2016/12/25.
    */
    import java.util.Arrays;

public class InsertSortDemo {
public static void main(String[] args) {
int[] A = {2,12,5,33,52,9};

    InsertSortDemo insertSortDemo = new InsertSortDemo();
    int[] sortedA = insertSortDemo.InsertSort(A);

    System.out.print(Arrays.toString(sortedA));
}

public int[] InsertSort(int[] numArray){
    //start from the second element,
    // because the very first element forms a sorted sequence
    for (int i=1;i<numArray.length;i++){
        int key=numArray[i];
        int j=i-1;

        while (j>=0 && numArray[j]>key){
            numArray[j+1]=numArray[j];
            j--;
        }
        numArray[j+1]=key;
    }
    return numArray;
}

}

##################################################
算法分析：红字部分就是插入排序的核心部分，比较简单，进行解释。
1、若第i个数是需要进行插入的，取已排好序的数组[1, i-1]；
2、在while循环处，使用[1,i-1]内的元素j与i元素进行对比，若比i元素大，则向后移动一格，若比i元素小或者j<0则退出循环；
3、由于退出循环的条件是j元素小于i元素，故j元素就是第一个比i元素小的元素，则i元素应该放在j+1的位置。为什么？因为原本的j+1元素比i元素大，已经向后复制了一个，故此时数组内有两个原j+1元素，直接将i元素放入j+1位置，正好可以去掉重复的j+1元素。同时考虑边界情况：j<0，如果j<0时退出了while循环，说明数组内的第0个元素都比i元素大，此时0元素已经被复制到1位置，又由于j<0退出while循环时必定是j=-1，所以此时的j+1位置就是0位置，可以直接放入i元素，不会出现越界。
