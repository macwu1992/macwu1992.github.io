这一部分代码做了一个web最基本的功能：
1）服务器的启动；
2）连接数据库；
3）部署app 在服务器上。
其中init，服务器的启动，数据库连接池的创建，都是协程。通过get_event_loop()来获取loop，用一启动init函数。
